# 건축개요 자동 생성기 (ArchOverview)

대지를 선택하고 기본 정보 몇 가지만 입력하면 **건폐율·용적률·연면적·법정 의무 면적이 자동으로 계산**되고, A4 규격의 건축개요 보고서로 바로 출력할 수 있는 웹 서비스입니다.

---

## 주요 기능

| 기능 | 설명 |
| :--- | :--- |
| 🗺️ 지도에서 대지 클릭 | OpenStreetMap(Leaflet.js) 무료 지도. 클릭하면 주소 자동 입력 |
| 🔍 주소 검색 | 검색창에 입력하면 지도가 해당 위치로 이동 (Nominatim 무료 API) |
| 🏗️ 전 용도지역 지원 | 준주거·1~3종 일반주거·전용주거·상업·공업·녹지 전 지역 법정 수치 자동 설정 |
| ⚡ 실시간 자동 계산 | 입력 즉시 건폐율·용적률·완화용적률·연면적·주차대수 재계산 |
| ⚖️ 법적 기준 검토 | 주차대수·주민공동시설·경로당·어린이집·조경면적 의무 기준 자동 비교 |
| 📄 PDF 보고서 출력 | 입력창 숨기고 A4 규격 건축개요 표만 깔끔하게 출력 |

## 실행 방법

### 1단계. Python 설치 확인
터미널(PowerShell)에서 실행:
```bash
python --version
```
> Python이 없다면 [python.org](https://www.python.org/downloads/)에서 설치 (설치 시 **"Add Python to PATH"** 필수 체크)

### 2단계. 필요 라이브러리 설치
```bash
pip install flask requests
```

### 3단계. 서버 실행
프로젝트 폴더에서:
```bash
python app.py
```

### 4단계. 브라우저 접속
```
http://127.0.0.1:5000
```

## 사용법

1. **① 대지 정보** 스텝: 주소 검색 또는 지도 클릭 → 용도지역 선택 → 대지면적 입력
2. **② 건물 기본 정보** 스텝: 층수, 세대 타입별 전용면적·세대수 입력
3. **③ 주차 & 부대시설** 스텝: 주차대수, 복리시설 면적 입력
4. **우측 탭 확인**: "건축개요 미리보기" 탭에서 자동 완성된 표 확인
5. **PDF 출력**: 우상단 "PDF 출력" 버튼 클릭

## 적용된 주요 계산식

| 항목 | 산식 |
| :--- | :--- |
| 완화용적률 | 기준용적률 + (1.5 × 기부채납면적 × 기준용적률) ÷ 실사용 대지면적 |
| 공급면적 | 전용면적 ÷ 전용률 (기본 74.8%) |
| 용적률 | 지상층 연면적 ÷ 대지면적 × 100 |
| 건폐율 | 건축면적(추정) ÷ 대지면적 × 100 |
| 법정 주차 | 85㎡이하: Σ(세대수×전용면적)/85, 85㎡초과: Σ(세대수×전용면적)/70 |
| 주민공동시설 | 100~1000세대: 세대수×2.5㎡ / 1000세대이상: 500+세대수×2㎡ |
| 기전실 | 지상층 연면적 × 4% (지하 산입) |
